# Alert

## Demos


### 基本用法 demo1
点击按钮发射五彩纸屑吧～异步触发只需要在 onClick 中返回一个 Promise 即可。

```jsx
import React from 'react';
import { AlertTest } from 'demo';

export default () => {
  return <AlertTest closable={true} content={'Here is an demo1'}></AlertTest>;
};

```

### 基本用法 demo2
demo2 点击按钮发射五彩纸屑吧～异步触发只需要在 onClick 中返回一个 Promise 即可。

```jsx
import React from 'react';
import { AlertTest } from 'demo';

export default () => {
  return <AlertTest closable={false} content={'Here is an demo2 '}></AlertTest>;
};

```


## 属性/Props

### InnerProps

向用户显示警告的信息时，通过警告提示，展现需要关注的信息。

|参数名|描述|类型|默认值|
|---|---|---|---|
|position|位置|string |`-`|
|size|尺寸|string |`-`|
|status|状态|'danger' \| 'error' \| 'success' |`-`|

### Alert

向用户显示警告的信息时，通过警告提示，展现需要关注的信息。

|参数名|描述|类型|默认值|版本|
|---|---|---|---|---|
|action|自定义操作项|React.ReactNode |`-`|2.15.0|
|closable|是否可以关闭|boolean |`false`|-|
