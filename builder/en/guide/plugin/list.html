# Official Plugins

  • @modern-js/builder-plugin-node-polyfill: Enable to use Node.js Polyfill, refer to webpack5's changelog (opens new window).
  • @modern-js/builder-plugin-esbuild：Use ESBuild as the transformer and minimizer to improve performance.
  • @modern-js/builder-plugin-swc: Use SWC as the transformer and minimizer to improve performance.

You can find their source code in modern.js/packages/builder (opens new window).
